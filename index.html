Advanced Excel-file Merger

Step 1

First you choose a "template" file that you want to work with.

Step 2

Select what cells you wish to gather data from and how you want them arranged in a new file

Step 3

Chose the files you want to merge according to your merging alternatives chosen during Step 2

Step 4

When you are satisfied with the result just go ahead and download the data as a new Excel file! Happy merging!

About this tool

This tool is meant to help with merging selective data from multiple files in order to save tidious time spent opening and closing files manually. It is currently in a BEATA mode with very limited options in order to make sure that it is beeing tested. Its future developement will be heavy influenced by user feedback to improve it's usefullness. Future plans to implement involves:

  • Decide what column or row to append the data as seen in the display window today
  • Making mathematical operations between cells as per user wishes
  • Include intelligens such as find the last value in a row or column instead of chosing a static cell
  • Overall design improvements on the site
